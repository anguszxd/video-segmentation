Linux 64bit binaries for running the computation of the local flow 
variance of an optical flow field as it is used in the motion segmentation

http://lmb.informatik.uni-freiburg.de/resources/binaries/pami2013_MoSeg.tar.gz

Usage
_________________________________________________________________

Use the Makefile to create the binary and run it given an 
optical flow file as the only argument:

example:
./cmpFlVar tennis454_ForwardFlow.flo
